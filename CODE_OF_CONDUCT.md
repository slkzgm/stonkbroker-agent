# Code of conduct

We want participation in this project to be safe, respectful, and productive.

Contributors must communicate constructively, respect differing viewpoints,
avoid harassment and discrimination, and focus criticism on ideas and code.
Private information, credentials, and vulnerability details must not be shared
without permission.

Project maintainers may edit or remove contributions and restrict participation
when behavior threatens the community or the security of users. Report conduct
concerns privately to the maintainers through the repository's Security tab.
